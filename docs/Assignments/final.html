Final Project: Multiple Linear Regression

Author

Dr. Joshua Lambert

Intro

Please review the lecture Introduction to Multiple Linear Regression. Ideally, your project will follow standard academic reporting guidelines. Please expand thoroughly during all three phases of your project.

Final Project Details and Guidelines

This academic project is a team effort involving a group of students, ideally with three-four members, unless specified otherwise by Dr. Lambert.

The main goal of the Final Project is to assess your understanding of Multiple Linear Regression (MLR). The project has three parts: a project proposal, a presentation, and a comprehensive paper, following the standard structure used in academic research, including Title page, Abstract, Introduction, Methods, Results, Discussion, and References.

The project proposal should be concise, focusing on the key ideas and excluding Results and Discussion. It lays the groundwork for the subsequent presentation. The presentation builds on the proposal by adding the omitted Results and Discussion sections, along with References. Feedback received during the presentation should be taken into account when finalizing the project paper.

The project paper is the most substantial part, requiring at least five pages. It involves a thorough exploration of the subject, with in-depth discussions tailored to the reviewer’s interests.

These components together simulate the academic research process, allowing participants to comprehensively utilize Multiple Linear Regression through collaboration and scholarly investigation.

For the project, you will develop a hypothesis and use NHANES data (specifically 2017-2018) to answer it. Present the findings in a clear and understandable manner, and consider incorporating insights from the lectures whenever possible.

Project Datasets: NHANES 2017-2018

For this project, you are required to use data from the National Health and Nutrition Examination Survey (NHANES) for the 2017-2018 cycle. You will access this data directly in R using the nhanesA package.

Using the nhanesA Package

First, ensure you have the package installed and loaded:

#| eval: false
install.packages("nhanesA")
library(nhanesA)

The 2017-2018 data cycle in NHANES is typically denoted by the suffix _J in the table names (e.g., DEMO_J for Demographics).

You can search for available tables in different categories (e.g., Demographics, Examination, Laboratory, Questionnaire) using nhanesTables. The year argument 2017 covers the 2017-2018 cycle.

#| eval: false
# List demographic tables for 2017-2018
nhanesTables('DEMO', 2017)

# List examination tables for 2017-2018
nhanesTables('EXAM', 2017)

To download a specific table, use the nhanes() function with the table name. For example, to get the Demographics data:

#| eval: false
# Download the 2017-2018 Demographics data
demo_data <- nhanes('DEMO_J')

If you need to analyze variables from multiple tables (e.g., comparing a health outcome from the Examination data across demographic groups), you will need to merge the datasets. NHANES tables usually contain a common identifier column SEQN.

#| eval: false
# Example of merging demographics with another table (e.g., Body Measures 'BMX_J')
bmx_data <- nhanes('BMX_J')
merged_data <- merge(demo_data, bmx_data, by = "SEQN")

Task: Identify variables suitable for a Multiple Linear Regression analysis. This typically involves a continuous outcome variable (Dependent Variable) and two or more predictor variables (Independent Variables), which can be continuous or categorical.

Information about the variables and what they measure can be found on the NHANES 2017-2018 website.

Project Proposal

  • Length: 1-2 pages.
  • Content:
    • Introduction: Briefly introduce the topic and the relevance of the variables you chose from NHANES.
    • Goals and Hypotheses: Clearly state the research question and hypothesis. Ensure it is testable via Multiple Linear Regression.
    • Statistical Analysis Plan (SAP): Outline the course of action for data collection (listing specific NHANES tables), data analysis, the type of statistical software used (R/RStudio), and plans for results and interpretations.
    • Meeting: Schedule a meeting with Dr. Lambert at least two weeks prior to the project presentation to discuss your proposal.

Project Presentation

  • Length: 20 minutes (10-15 PowerPoint slides).
  • Content:
    • Structure: One slide for each section from your paper (hypothesis, data description, statistical methods, results, and conclusions). Distribute these slides evenly throughout the presentation, adding detailed explanations in between.
    • Creativity: Use original and engaging methods to present your results. Utilize external resources and materials to enhance your presentation.
    • Q&A: Reserve 5-10 minutes for questions.
    • Peer Review: Classmates will critique each presentation based on a scoring rubric. These reviews will be averaged for the peer review grade.

Project Paper

  • Length: At least 5 pages.
  • Content:
    • Introduction: Provide a comprehensive introduction to the topic and explain why this research is necessary and interesting.
    • Data Description: Present the dataset (NHANES 2017-2018) and give a short description of the variables.
    • Statistical Methods: Justify the chosen statistical methods (Multiple Linear Regression) and explain how they will be applied (checking assumptions such as linearity, normality of residuals, homoscedasticity, multicollinearity, etc.).
    • Results: Present the findings from the analysis. Report statistical results as regression coefficients (95% CI; p-value), R², and adjusted R² where appropriate.
    • Discussion and Conclusion: Summarize the findings, discuss implications, and suggest future research directions. Address possible pitfalls in the study.
    • References: Include a comprehensive list of references used in your research.
